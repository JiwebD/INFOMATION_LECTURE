ware : 제품
hardware : CPU, RAM, DISK... 부품
software : hardware 자원을 이용해서 특정 목적달성위해 사용되는 컴퓨터 프로그램

  System Software(Operating System)
  H/W자원 관리 -> os(운영체제) -> os의 도움을 받아서 application 실행
  hardware자원을 관리 / Application 관리 / file 관리...

  Application : 사용자 요구사항을 만족하는 프로그램



미들웨어    

OS(운영체제)와 application 사이에 끼워넣어 사용하는 프로그램.

 ex) jvm(Java Virtual Machine)
