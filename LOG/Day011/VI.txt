VI
Unix 및 Linux 운영 체제에서 사용되는 텍스트 편집기.

-------------------------------------------------
vi 모드	-	vi /파일경로
-------------------------------------------------
이동 모드	(초기모드)	- 파일 내 이동 
입력 모드	(편집모드)	- 파일 내 수정 
EX 모드(명령어 실행) 	- 파일 내외에 특정 명령어 실행



이동 모드 
커서이동 : h,j,k,l
단어이동 : w,e,b
행이동 : 0, $
페이지이동 : ctrl + f, b
마지막, 처음 : gg, G
삭제 : x, dd
복사/붙여넣기 : yy, p




입력모드 : a, i, o, s




실행모드

나가기
q : 나가기
wq : 저장하고 나가기
q! : 저장하지않고 나가기



행 넘버 표시 : set number
행 넘버 표시 취소 : set nonumber
