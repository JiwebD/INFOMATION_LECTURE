1. 물리계층
역할 : 장치간 물리적 연결
약속 : 디지털 신호 전달
장치 : 케이블, 리피터, 허브

2. 데이터링크 계층
역할 : Point to Point장치간 데이터 운송방식
약속 : LAN(이더넷) WAN(HDLC,PPP)
장치 : L2 SWITCH  (충돌방지)

3. 네트워크 계층
역할 : 경로탐색
약속 : IP, ICMP, ARP, Routing Protocol
장치 : Router

4. 전송계층
역할 : End to End종단간(송, 수신지간) 데이터의 신뢰성을 보장한다.
약속 : TCP / UDP  ☆ 시험
장치 : L4 Switch(로드밸런서)
